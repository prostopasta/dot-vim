# dot-vim
My dot-files config repository for ViM and ShellCheck

# What it does
1. Install `vim-enhanced`
1. Install `shellcheck`

# Supporting distros
1. RHEL-based
1. Debian-based

# Installation
Just copy the repository to your machine and execute `install.sh` script:
```sh
git clone <REPO_URL>
cd dot-vim/ && bash install.sh
```
